Relaciones entre Clases

Uso

Una clase usa "usa" a otra clase cuando alguno de sus atributos o algunos de los parámetros o resultados de sus métodos son objetos de otra clase.

Ejemplo:

        class Persona{}

        class Factura{
            setCliente(cliente){ //Si cliente es un objeto de tipo Persona, entonces la clase Factura "usa" a Persona.
                this.cliente = cliente 
            }
        }

Herencias

Las herencias nos permite asociar un objeto prototipo con un método constructor. Así, el objeto hereda las propiedades del objeto.
