Declaración de variables

Declaramos las variables en JavaScript con: var, let o const.

        var num1
        let num2
        const NUM3

Diferencias

vardeclara las variables a nivel "funcional". Es decir, el ámbito de la variable es toda la función en la que se declara y sus bloques internos.

Si estamos al máximo nivel, la variables es global

Usando var, la declaración va al principio de la función.

letdeclara las variables a nivel "de bloque". Es decir, el ámbito de la variable es todo el bloque en el que se declara y sus bloques internos.

Si estamos al máximo nivel, la variables no entra en la función

constdeclara las variables a nivel "funcional". Es decir, igual que var

Pero no se puede "reasignar".

Moraleja: Si puedes utiliza const. Si no, usa let salvo que necesites una variable global (usas var).
