Funciones

Declaramos las variables en JavaScript con la palabra reservada function.

Las funciones tienen:

  1. Nombre.
  2. Parámetros.
  3. Resultado.
  4. Cuerpo.
        function nombre(parametro1, parametro2){
            //cuerpo con el código de la función.
            return resultado
        }

Llamamos a las funciones por su nombre y dándole los parametros. No olvidamos "recoger" el resultado.

Ejercios de Funciones

Estrategia para programar Funciones

  1. Hacemos la DECLARACIÓN:
    1. Elegir el nombre de la función.
    2. Elegir el nombre de los parámetros.
    3. Identificando el tipo de dato del resultado.
  2. PROBAMOS la función
    • Devolvemos un resultado "mock" (42).
    • Llamamos a la función sin parámetros y comprobamos que obtenemos el resultado mock.
  3. BUSCAMOS TESTS (ejemplos).
  4. PROGRAMAMOS el cuerpo de la función hasta que pasen los test.
  5. Si el problema es complicado, lo "simplifico".
